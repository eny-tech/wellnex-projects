new dedicated org, `wellnexsolutionsai`, using a family-prefix naming scheme:

| Old (this repo)              | New                                      |
| ---------------------------- | ---------------------------------------- |
| eny-tech/wellnex             | wellnexsolutionsai/product-web           |
| eny-tech/wellnex-server      | wellnexsolutionsai/product-api           |
| eny-tech/wellnex-health-agents | wellnexsolutionsai/product-health-agents |
| eny-tech/wellnex-agents      | wellnexsolutionsai/platform-dev-agents   |
| eny-tech/wellnex-docs        | wellnexsolutionsai/governance-docs       |
| eny-tech/wellnex-projects    | wellnexsolutionsai/governance-projects   |

Full git history (all branches and tags) was mirror-pushed to the new repo, so
nothing is lost — please update your remotes:

```sh
git remote set-url origin git@github.com:wellnexsolutionsai/governance-projects.git
```
